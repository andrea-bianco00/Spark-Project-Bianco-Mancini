This repository contains the final project developed for the course **IT Tools 2 (NoSQL, Big Data Processing with Spark)**.

The project focuses on large-scale data processing using **Apache Spark**.

---

## Project Content

The main components of the repository are:

- **Spark_Project_Bianco_Mancini.ipynb**  
  Main Jupyter notebook implementing the complete Spark workflow:
  data loading, preprocessing, distributed analysis, and result generation.

- **config.yaml**  
  Global configuration file used to define data paths, time ranges, and remote data sources.  
  If this file is modified, the notebook must be re-run from the beginning.

- **departements.geojson**  
  Geographical data used for spatial analysis and mapping.

- **html/**  
  Folder containing HTML files generated during the execution of the notebook, representing final visual outputs.
